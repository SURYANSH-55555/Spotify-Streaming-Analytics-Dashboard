# 🎵 Spotify Streaming Analytics Dashboard

![Dashboard Preview](Spotify%20Dashboard%20preview.png)

## 📖 Project Overview

The Spotify Streaming Analytics Dashboard is an interactive Business Intelligence solution developed using **Power BI** to analyze Spotify music streaming data and uncover valuable insights about tracks, artists, release trends, and audio characteristics.

The dashboard enables users to explore streaming performance, identify popular tracks and artists, understand music composition patterns, and analyze relationships between various audio features through interactive visualizations.

This project demonstrates end-to-end data analytics skills including:

- Data Cleaning & Transformation
- Data Modeling
- Power Query
- DAX Measures
- Interactive Dashboard Design
- Data Visualization
- Business Intelligence Reporting

# 🔍 Dashboard Features

## 1. KPI Overview

The dashboard begins with high-level performance indicators.

### Total Streams

Displays the cumulative streams across all tracks available in the dataset.

**Total Streams:** 383 Billion+

### Average Danceability

Measures how suitable a track is for dancing based on rhythm, beat, and tempo.

**Average Score:** 67.41%

### Average Energy

Represents the intensity and activity level of songs.

**Average Score:** 64.36%

### Average Valence

Indicates the positivity or happiness conveyed by a song.

**Average Score:** 51.17%
